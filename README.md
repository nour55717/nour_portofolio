## Projects Included

### Intelligent Traffic Surveillance & Object Analysis System

A computer vision system built using YOLOv8 and OpenCV for detecting and tracking people, vehicles, and bicycles in real-time video scenes.

### Medical Image Registration using PSO

A medical imaging project that uses Particle Swarm Optimization to align CT/MRI images and evaluate registration quality using different similarity metrics.

### Expert Heart Disease Risk Predictor

A hybrid clinical decision-support system combining machine learning with an expert reasoning layer to predict heart disease risk.

### King County House Price SVR Predictor

A regression-based real estate prediction system using SVR, preprocessing, feature engineering, PCA, and Streamlit visualization.

### Operating Systems Algorithms GUI Simulator

A desktop GUI application that simulates CPU scheduling and memory management algorithms with interactive visualizations.

### Healthcare Risk Analysis using CCA

A statistical analysis project using Canonical Correlation Analysis to study relationships between lifestyle and clinical health indicators.

### Spam vs Ham Message Analysis

A text classification and visualization project analyzing spam and ham messages using NLP techniques, word clouds, and feature exploration.

### Excel Interactive Dashboards

Business intelligence dashboards created using Excel Pivot Tables, charts, slicers, and timeline filters.
